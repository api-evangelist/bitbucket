Atlassian has introduced immutable tags for the Bitbucket Packages container registry, allowing workspace administrators to lock container image tags after their initial push to prevent overwrites or modifications. This feature addresses deployment predictability and compliance concerns by ensuring production and release tags consistently point to tested and approved images. Workspace admins can enable immutable tags per-registry through Bitbucket settings.
