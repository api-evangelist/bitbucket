# Bitbucket (bitbucket)
Bitbucket is a Git-based source code repository hosting service owned by Atlassian. It offers both commercial plans and free accounts with unlimited private repositories.

**URL:** [Visit APIs.json URL](https://bitbucket.org)

## Tags:

## APIs

### Bitbucket Cloud REST API
The REST API for Bitbucket Cloud allows you to build apps using any language you want. It exposes a wide range of primitive operations to perform on resources like repositories, changesets, pull requests, and users.

**Human URL:** [https://developer.atlassian.com/bitbucket/api/2/reference/](https://developer.atlassian.com/bitbucket/api/2/reference/)

### Bitbucket Webhooks API
Webhooks provide a way to configure Bitbucket Cloud to make requests to your server when certain events occur in a repository.

**Human URL:** [https://developer.atlassian.com/cloud/bitbucket/rest/api-group-webhooks/](https://developer.atlassian.com/cloud/bitbucket/rest/api-group-webhooks/)
